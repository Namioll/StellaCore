# StellaCore

Легаси-ядро для Minecraft-сервера на базе Spigot, изначально разработанное для ныне закрытого проекта.

> **Важно:** Данному проекту более полугода, он написан на устаревшем фреймворке Spigot (не Paper) с использованием Maven. Этот код не отражает мой текущий уровень навыков и подходы к разработке. Репозиторий выложен исключительно в архивных и ознакомительных целях.

## Описание

StellaCore — это комплексный плагин-ядро, расширяющий возможности сервера Minecraft. Он добавляет кастомизированную систему чата, магические предметы, набор команд для администрации и модерации, а также инструменты для стримеров.

## Возможности

- **Кастомный чат**
  - Индивидуальные префиксы, титулы, цвета ника и сообщений для каждого игрока.
  - Локальный и глобальный режимы чата (префикс `!` для отправки сообщения на весь сервер).
  - Стример-режим с особым форматированием чата и отдельным меню.
  - Фильтр нецензурной лексики с настраиваемым списком запрещённых слов.
  - Кликабельные ссылки на Twitch в чате.

- **Магические предметы**
  - Волшебная палочка, магическая кость, посох и мино-кость.
  - Элементы RPG: прокачка волшебной палочки, ее сила растет с каждым уровнем.
  - Все свойства предметов настраиваются через конфигурационные файлы.
  - Команда для выдачи предметов с возможностью указать уровень.

- **Команды для профессий и модерации**
  - `/sud` – призвать игрока на суд (для судьи).
  - `/menu` – открыть меню судьи (Sudia).
  - `/mc` – отправить объявление мэра всему серверу.
  - `/alarm` – отправить системное уведомление (например, о перезапуске).
  - `/pygalka` – запустить ивент с "внезапной темнотой" на сервере.
  - `/chatreload` – перезагрузить настройки чата без перезапуска сервера.

- **Инструменты для стримеров**
  - `/stream` – открыть меню стримера и оповестить сервер о начале стрима.
  - `/link` – установить или просмотреть ссылку на Twitch-канал (отображается в чате как кликабельная).
  - `/banwords` – просмотреть список запрещённых слов.

| Команда | Право (Permission) | Описание |
|---------|-------------------|----------|

| `/link [ссылка]` | `stellacore.stream` | Установить или посмотреть ссылку на Twitch. |

## Установка

1. Забилдите плагин с помощью Maven.
2. Поместите JAR-файл в папку `plugins` вашего сервера.
3. Убедитесь, что на сервере установлен и настроен ItemsAdder — плагин зависит от него.
4. Запустите или перезапустите сервер.
5. Настройте плагин через YAML-файлы в папке `plugins/StellaCore`:
   - `chat.yml` – настройки чата для игроков (префиксы, цвета, статус стримера и т.д.).
   - `banwords.yml` – список слов, которые будут фильтроваться в чате.
   - `magicwand.yml` – свойства магических предметов.

## Конфигурация

При первом запуске плагин создаёт три конфигурационных файла:

- **chat.yml** – хранит данные игроков: префикс, титул, цвет ника, цвет сообщений, ссылку на Twitch, статус стримера, разрешённые цвета и поддержку hex-кодов.
- **banwords.yml** – содержит список слов, заменяемых на звёздочки в чате.
- **magicwand.yml** – задаёт свойства (название, описание, зачарования и т.д.) для каждого магического предмета.
